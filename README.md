# CS2-Simplify-HUD-Settings
Simplify ur HUD!

<p>Download the config and put it: <pre>...\steamapps\common\Counter-Strike Global Offensive\game\csgo\cfg</pre></p>

<h3>The Settings:</h3>
<blockquote>
  <i>HUD[0 or 1]</i> - <code>Hide/Show HUD (togglable)</code>
  <br><i>UCol[num]</i> - <code>Color of HUD (0 to 11)</code>
  <br><i>UDTarget[0 or 1]</i> - <code>Shows enemy's name when you aim at him (togglable)</code>
  <br><i>UPAva[0 or 1] / UPAvatar[0 or 1]</i> - <code>Display avatars as number(toggleable)</code>
  <br><i>UScale[num]</i> - <code>Scale of HUD (0.5 to 2, only +0.05)</code>
  <br><i>FPS[0 or 1]</i> - <code>Toggle fps count (togglable)</code>
</blockquote>
<h3>Teammates</h3>
<blockquote>
  <i>NMark[num]</i> - <code>Fading teammate's overhead marker near to crosshair (+0.1 only, 0 to 1)</code>
  <p>
  <br><i>TEquip[num]</i> - <code>Showing mate's equipment & other (0 / 1 / 2 / 3)</code>
  <br>0 - <i><b>name and marker</b></i> / 1 - <i><b>marker only</b></i> / 2 - <i><b>name, hp and marker</b></i> / 3 - <i><b>weapon, hp, name and marker</b></i></p>
  
  <i>TACol[0 or 1]</i> - <code>Colored mate's avatars (togglable)</code>
  <br><i>TECol[0 or 1]</i> - <code>Colored mate's names (togglable)</code>
  <br><i>AAnim[0 or 1]</i> - <code>Enable animated avatars (togglable)</code>
</blockquote>
<h3>Other things</h3>
<blockquote>
  <i>EMute</i> - <code>Enemy team mute (togglable)</code>
</blockquote>
